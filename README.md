## アドバイス

- "わからない"、"できない" はどこかの誰かも同じだと思い、手がかりをいつか公開しよう。
- "わからない"、"できない" は筋肉痛と同じ。むしろワクワクしよう。「喜べ。共有できるネタを見つけたのだ。」
